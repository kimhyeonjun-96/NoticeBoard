# 스프링 시큐리티를 이용한 게시판
## 개발기간 
- 24.02.20 ~ 24.02.28


## 개발 환경
- java17
- **IDE** : intelliJ
- **Framework** : Springboot(3.x)
- **Database** : Mysql(8.1.0)
- **ORM** : JPA



## 주요기능
#### 로그인
- 로그인 시 스프링 시큐리티로 검증으로 로그인


#### 회원가입( 회원, 수의사 )
- 회원가입시 스프링 시큐리티 회원가입
- ID 중복 체크(완료)


#### 글
- 글 작성
- 글 삭제
- 글 수정
- 전체 글 리스트 확인


#### 회원 마이페이지
- 회원 정보 변경
- 회원 탈퇴
- 회원이 작성한 글 확인
